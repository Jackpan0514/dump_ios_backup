# dump ios backup
dump ios domain file from itunes backup

## require
1. macosx & itunes backup
2. lua5.3

## tutorial
```
$ make
$ lua dump_backup.lua <your itunes backup> [domain]
```
## domain example
domain | info
:-----:|:-------
AppDomain-com.tencent.xin | 微信 
AppDomain-com.tencent.QQMusic | QQ音乐
AppDomain-com.tencent.mqq | 手机QQ
